# vinci-repo

## git commands

download repository
```
git clone _GIT_REPO_URL_
```

check local git changes
```
git status
```

add changes ready for commit to local git system
```
git add .
```

commit changes
```
git commit -m "commit message"
```

push to remote git repository
```
git push origin _BRANCH_
```

switch branch
```
git checkout -b _BRANCH_
```
